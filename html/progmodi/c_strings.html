01 — Fundamentos

String = Vetor de char

Em C não existe um tipo "string" nativo. Um texto é simplesmente um array (vetor) de caracteres terminado por um caractere especial: \0 (null terminator — valor ASCII 0).

String
Sequência de caracteres armazenada como um array de char, terminada pelo caractere nulo \0.
Array / Vetor
Conjunto de elementos do mesmo tipo organizados em posições consecutivas na memória. Cada posição é acessada por um índice.
Null Terminator (\0)
Caractere com valor ASCII 0. Marca o fim da string. Sem ele, funções como printf leriam memória além do texto.
Índice
Número que indica a posição de um elemento no array. Em C, sempre começa em 0.
Tamanho do buffer
Quantidade total de char reservados, incluindo o espaço para o \0. Deve ser pelo menos comprimento + 1.
Literal de string
Texto entre aspas duplas: "Olá". O compilador adiciona o \0 automaticamente.

string_conceito.c 
#include <stdio.h>

int main() {

    /* Declaração — 3 formas equivalentes */
    char a[] = "Oi, C!";          // tamanho automático (7)
    char b[20] = "Oi, C!";        // buffer maior — mais seguro
    char c[7] = {'O','i',',',' ','C','!','\0'};  // char por char

    printf("%s\n", a);   // %s imprime a string inteira

    /* Acessar caractere individual */
    printf("Primeiro char: %c\n", a[0]);   // 'O'
    printf("Quinto char:   %c\n", a[4]);   // 'C'

    /* Modificar caractere — strings mutáveis em array */
    a[0] = 'E';
    printf("%s\n", a);   // "Ei, C!"

    /* Iterar por cada caractere com for */
    for (int i = 0; a[i] != '\0'; i++) {
        printf("[%d]='%c' ", i, a[i]);
    }
    printf("\n");

    return 0;
}
⚠️
Buffer overflow — Se você declara char nome[10] e guarda mais de 9 caracteres (+ \0), você invade memória de outras variáveis. Sempre declare um buffer maior do que o esperado.
💡
String literal vs array — char s[] = "Oi" cria um array editável. char *s = "Oi" aponta para uma área somente-leitura — modificar via ponteiro causa comportamento indefinido.
02 — Português de Verdade

Acentuação & Locale

Para exibir ã, ç, é, õ corretamente no terminal Windows/Linux/macOS é preciso configurar o locale e usar a codificação certa. Duas bibliotecas resolvem isso.

setlocale()

Define o locale do programa — idioma, codificação e formatação de números/datas. Para português brasileiro use:

setlocale(LC_ALL, "pt_BR.UTF-8");

No Windows também funciona com "Portuguese_Brazil.1252".

LC_ALL vs LC_CTYPE

LC_ALL afeta tudo. LC_CTYPE afeta apenas classificação de caracteres (islower, toupper, etc.). Para textos em PT-BR, LC_ALL é a escolha mais segura.

wchar_t (Wide Char)

Tipo alternativo para caracteres Unicode. Usa 2 ou 4 bytes. Strings wide usam L"texto" e funções como wprintf, fgetws. Mais complexo; prefira UTF-8 com locale.h quando possível.

UTF-8 no terminal

Salve o arquivo .c como UTF-8 no editor. No Windows, execute chcp 65001 no prompt antes de rodar o programa, ou use Windows Terminal que já suporta UTF-8.

locale_ptbr.c 
#include <stdio.h>
#include <locale.h>    // necessário para setlocale()

int main() {

    /* Configura locale para Português do Brasil (UTF-8) */
    setlocale(LC_ALL, "pt_BR.UTF-8");

    char nome[60];
    printf("Digite seu nome (pode usar acentos): ");
    fgets(nome, sizeof(nome), stdin);

    printf("Olá, %s!\n", nome);
    printf("Atenção: acentos funcionam com UTF-8.\n");
    printf("Ação concluída com êxito!\n");

    return 0;
}
💡
Compilando com UTF-8 no GCC (Windows): gcc -o programa programa.c -finput-charset=UTF-8 -fexec-charset=UTF-8 No Linux/macOS o terminal já é UTF-8 por padrão — basta o setlocale.

03 — Entrada de Dados

Lendo Strings do Usuário

Há várias formas de ler strings. Cada uma tem limitações importantes — escolher a errada pode travar o programa ou criar falhas de segurança.

scanf
%s
Lê até espaço
Para apenas no 1° espaço — não lê nomes compostos. Risco de overflow sem limite.
fgets
Lê a linha inteira ✓
Aceita espaços. Respeita o tamanho do buffer. Inclui '\n' no final.
scanf
%Ns
Lê até N chars
scanf("%19s", nome) limita a 19 chars. Ainda para no espaço.
gets
NUNCA use! ✗
Sem limite de tamanho. Vulnerabilidade clássica. Removida do C11.
🔴
gets() está banida do C moderno — Compiladores modernos emitem aviso ou erro. Substitua sempre por fgets(buf, tamanho, stdin).
leitura_strings.c 
#include <stdio.h>
#include <string.h>   // strlen, strcspn
#include <locale.h>

int main() {
    setlocale(LC_ALL, "pt_BR.UTF-8");

    char nome[60];
    char cidade[40];

    /* ── fgets: lê linha inteira (com espaços) ── */
    printf("Nome completo: ");
    fgets(nome, sizeof(nome), stdin);

    /* fgets mantém o '\n' — remover com strcspn */
    nome[strcspn(nome, "\n")] = '\0';

    /* ── scanf %s: lê até espaço ── */
    printf("Cidade (sem espaço): ");
    scanf("%39s", cidade);   // 39 = tamanho - 1

    printf("\nNome:   %s\n", nome);
    printf("Cidade: %s\n", cidade);
    printf("Chars no nome: %zu\n", strlen(nome));

    return 0;
}
💡
Por que strcspn(nome, "\n")?
fgets guarda o \n (Enter) que o usuário pressionou dentro do array. strcspn retorna o índice do primeiro \n encontrado — substituí-lo por \0 corta a string no lugar certo.
⚠️
Misturar scanf e fgets — Se você usar scanf antes de fgets, o \n do Enter anterior fica no buffer de entrada e o fgets "lê" uma linha vazia. Solução: adicione getchar(); ou scanf(" "); (espaço antes) para consumir o \n pendente.
04 — Trabalhando com Texto

Manipulando Strings

Strings em C são arrays, então não se copiam ou comparam com = ou ==. É preciso usar funções específicas ou percorrer o array caractere por caractere.

🔴
O erro clássico — nunca faça:
str1 = str2; ← copia o ponteiro, não o conteúdo!
if (str1 == str2) ← compara endereços, não o texto!

Iterando manualmente

iteracao.c 
#include <stdio.h>
#include <locale.h>

int main() {
    setlocale(LC_ALL, "pt_BR.UTF-8");

    char frase[] = "hello world";

    /* ── Contar caracteres manualmente ── */
    int len = 0;
    while (frase[len] != '\0') len++;
    printf("Comprimento manual: %d\n", len);

    /* ── Converter para MAIÚSCULO manualmente ── */
    for (int i = 0; frase[i] != '\0'; i++) {
        if (frase[i] >= 'a' && frase[i] <= 'z') {
            frase[i] -= 32;   // diferença ASCII entre 'a' e 'A'
        }
    }
    printf("Maiúsculo: %s\n", frase);  // HELLO WORLD

    /* ── Inverter string ── */
    int ini = 0, fim = len - 1;
    while (ini < fim) {
        char tmp = frase[ini];
        frase[ini] = frase[fim];
        frase[fim] = tmp;
        ini++; fim--;
    }
    printf("Invertido: %s\n", frase);  // DLROW OLLEH

    return 0;
}

ctype.h — classificar e converter caracteres

📦
#include <ctype.h> — fornece funções para testar e converter caracteres individuais sem precisar comparar com valores ASCII manualmente.
isalpha
É letra? (a-z / A-Z)
isalpha('A') → 1 (true)
isdigit
É dígito? (0-9)
isdigit('7') → 1 (true)
isspace
É espaço/tab/\n?
isspace(' ') → 1 (true)
toupper
Converte para maiúsculo
toupper('a') → 'A'
tolower
Converte para minúsculo
tolower('Z') → 'z'
isalnum
É letra ou dígito?
isalnum('3') → 1 (true)
ctype_uso.c 
#include <stdio.h>
#include <ctype.h>    // toupper, tolower, isdigit...

int main() {
    char frase[] = "ola, Mundo! 123";

    /* Contar letras, dígitos e espaços */
    int letras = 0, digitos = 0, espacos = 0;
    for (int i = 0; frase[i]; i++) {        // frase[i] != '\0' implícito
        if      (isalpha(frase[i])) letras++;
        else if (isdigit(frase[i])) digitos++;
        else if (isspace(frase[i])) espacos++;

        /* Converter tudo para maiúsculo */
        frase[i] = (char) toupper(frase[i]);
    }

    printf("Resultado: %s\n", frase);
    printf("Letras: %d | Dígitos: %d | Espaços: %d\n",
           letras, digitos, espacos);

    return 0;
}
05 — Biblioteca Padrão

Funções de string.h

A biblioteca #include <string.h> fornece as operações mais comuns sobre strings — copiar, concatenar, comparar, buscar. Todas esperam que a string seja terminada em \0.

Função Biblioteca O que faz
strlen(s) string.h Retorna o comprimento da string (sem contar o \0). size_t strlen(const char *s)
strcpy(dest, src) string.h Copia src para dest (dest deve ter espaço suficiente!). char *strcpy(char *dest, const char *src)
strncpy(dest, src, n) string.h Copia no máximo n caracteres. Mais seguro que strcpy. char *strncpy(char *dest, const char *src, size_t n)
strcat(dest, src) string.h Concatena src ao final de dest. dest deve ter espaço para ambos + \0. char *strcat(char *dest, const char *src)
strncat(dest, src, n) string.h Concatena no máximo n chars de src. Sempre prefira à strcat. char *strncat(char *dest, const char *src, size_t n)
strcmp(s1, s2) string.h Compara duas strings. Retorna 0 se iguais, negativo se s1 < s2, positivo se s1 > s2. int strcmp(const char *s1, const char *s2)
strncmp(s1, s2, n) string.h Compara até n caracteres. Útil para prefixos. int strncmp(const char *s1, const char *s2, size_t n)
strchr(s, c) string.h Retorna ponteiro para a 1ª ocorrência do char c em s, ou NULL se não encontrado. char *strchr(const char *s, int c)
strstr(s, sub) string.h Retorna ponteiro para a 1ª ocorrência de sub dentro de s, ou NULL. char *strstr(const char *s, const char *sub)
sprintf(buf, fmt, ...) stdio.h Formata e escreve numa string (como printf, mas para char[]). int sprintf(char *buf, const char *fmt, ...)
sscanf(s, fmt, ...) stdio.h Lê e extrai valores de uma string (como scanf, mas de char[]). int sscanf(const char *s, const char *fmt, ...)
string_h_uso.c 
#include <stdio.h>
#include <string.h>
#include <locale.h>

int main() {
    setlocale(LC_ALL, "pt_BR.UTF-8");

    char nome[30] = "Maria";
    char sobrenome[] = " Silva";
    char copia[30];

    /* strlen — comprimento */
    printf("Comprimento: %zu\n", strlen(nome));  // 5

    /* strcpy — copiar */
    strcpy(copia, nome);
    printf("Cópia: %s\n", copia);  // Maria

    /* strncat — concatenar com segurança */
    strncat(nome, sobrenome, sizeof(nome) - strlen(nome) - 1);
    printf("Nome completo: %s\n", nome);  // Maria Silva

    /* strcmp — comparar strings */
    if (strcmp(nome, "Maria Silva") == 0) {
        printf("Strings iguais!\n");
    }

    /* strchr — buscar caractere */
    char *pos = strchr(nome, 'S');
    if (pos) printf("'S' encontrado em: %s\n", pos);  // Silva

    /* strstr — buscar substring */
    if (strstr(nome, "Silva")) {
        printf("Substring encontrada!\n");
    }

    /* sprintf — montar string formatada */
    char msg[80];
    int idade = 22;
    sprintf(msg, "Nome: %s, Idade: %d", nome, idade);
    printf("%s\n", msg);

    return 0;
}
⚠️
strcpy e strcat sem limite são perigosas — Use sempre strncpy e strncat em código real, passando o tamanho disponível no buffer de destino. A expressão sizeof(dest) - strlen(dest) - 1 calcula o espaço restante.

sprintf & sscanf — strings como streams

sprintf_sscanf.c 
#include <stdio.h>

int main() {

    /* sprintf — construir string a partir de dados */
    char data[20];
    int  dia = 7, mes = 9, ano = 2025;
    sprintf(data, "%02d/%02d/%d", dia, mes, ano);
    printf("Data formatada: %s\n", data);  // 07/09/2025

    /* sscanf — extrair valores de uma string */
    char entrada[] = "42 3.14 Brasil";
    int   n;
    float f;
    char  pais[20];
    sscanf(entrada, "%d %f %19s", &n, &f, pais);
    printf("Inteiro: %d | Float: %.2f | País: %s\n", n, f, pais);

    return 0;
}
06 — Pratique

Exercícios

Coloque em prática tudo que aprendeu sobre strings, leitura segura e manipulação.

EX 01 Conta Vogais
  • Leia uma frase com fgets.
  • Percorra a string com um for até \0.
  • Conte quantas vogais (a, e, i, o, u — maiúsculas e minúsculas) existem.
  • Exiba o total.
💡 Dica: use tolower(frase[i]) antes de comparar para tratar maiúsculas e minúsculas de uma vez.
EX 02 Palíndromo
  • Leia uma palavra com scanf("%s").
  • Use strlen para saber o tamanho.
  • Compare caracteres do início e do fim simultaneamente (dois índices: ini e fim).
  • Diga se a palavra é um palíndromo (arara, radar, ovo).
💡 Dica: while (ini < fim) — se algum par de chars for diferente, já não é palíndromo.
EX 03 Cadastro com Nome e Cidade
  • Configure o locale para pt_BR.UTF-8.
  • Leia nome completo (com espaços) e cidade com fgets.
  • Remova o \n de ambos com strcspn.
  • Use sprintf para montar uma string formatada: "Bem-vindo, [nome]! Cidade: [cidade]."
  • Exiba a string montada.
💡 Dica: declare um buffer grande o suficiente para a mensagem montada: char msg[120];
EX 04 Substituir Caractere
  • Leia uma frase com fgets.
  • Leia um caractere para buscar e outro para substituir.
  • Percorra a string e substitua todas as ocorrências.
  • Exiba a frase modificada.
💡 Dica: leia os chars com scanf(" %c", &busca); — o espaço descarta o \n pendente do fgets anterior.
EX 05 Validador de Senha
  • Leia uma senha com fgets (máximo 30 chars).
  • Usando funções de ctype.h, verifique se a senha tem:
  • — ao menos 8 caracteres (use strlen)
  • — ao menos 1 letra maiúscula (isupper)
  • — ao menos 1 dígito (isdigit)
  • Exiba quais critérios foram ou não atendidos.
💡 Dica: use variáveis int temMaiuscula = 0, temDigito = 0; e defina-as como 1 quando encontrar o caractere correspondente no loop.

Gabarito — Exercício 01 (conta vogais)

conta_vogais.c 
#include <stdio.h>
#include <string.h>
#include <ctype.h>
#include <locale.h>

int main() {
    setlocale(LC_ALL, "pt_BR.UTF-8");

    char frase[100];
    printf("Digite uma frase: ");
    fgets(frase, sizeof(frase), stdin);
    frase[strcspn(frase, "\n")] = '\0';

    int vogais = 0;
    for (int i = 0; frase[i]; i++) {
        char c = (char) tolower(frase[i]);
        if (c=='a' || c=='e' || c=='i' || c=='o' || c=='u')
            vogais++;
    }

    printf("Vogais encontradas: %d\n", vogais);
    return 0;
}
