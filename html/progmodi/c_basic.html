01 — Fundamentos

Variáveis & Tipos

Uma variável é uma caixa com nome na memória do computador. Ela guarda um valor que pode mudar durante o programa.

Variável
Espaço nomeado na memória que armazena um valor.
Tipo de Dado
Define o que a variável pode guardar (número, texto, etc.) e quanto espaço ocupa.
Declaração
Momento em que se cria a variável no código: tipo nome;
Inicialização
Atribuir um valor na hora da declaração: int x = 5;
Escopo
Região do código onde a variável existe e pode ser usada.
printf / scanf
Funções para imprimir e ler valores do usuário no terminal.
Tipo Tamanho O que guarda Exemplo Formato printf
int 4 bytes Inteiros (sem vírgula) 42, -7, 0 %d
float 4 bytes Decimais (precisão simples) 3.14f, -0.5f %f
double 8 bytes Decimais (precisão dupla) 3.141592653 %lf
char 1 byte Um único caractere 'A', 'z', '5' %c
variaveis.c 
#include <stdio.h>

int main() {

    /* ── Declaração e inicialização ── */
    int    idade       = 20;
    float  nota        = 8.5f;
    double pi          = 3.14159265;
    char   inicial     = 'A';

    /* ── Exibir valores ── */
    printf("Idade:   %d\n",  idade);
    printf("Nota:    %.1f\n", nota);
    printf("Pi:      %lf\n", pi);
    printf("Inicial: %c\n",  inicial);

    /* ── Ler do usuário ── */
    printf("Digite sua idade: ");
    scanf("%d", &idade);   // & = endereço de memória
    printf("Você tem %d anos.\n", idade);

    return 0;
}
💡
Por que & no scanf? — O & passa o endereço da variável, para que scanf saiba onde na memória escrever o valor digitado pelo usuário.
02 — Cálculos & Comparações

Operadores

Operadores são símbolos que realizam operações sobre valores e variáveis. Existem três grupos principais.

Aritméticos — fazem contas

+
Adição
a + b → soma
-
Subtração
a - b → diferença
*
Multiplicação
a * b → produto
/
Divisão
10 / 3 → 3 (int!)
%
Módulo (resto)
10 % 3 → 1
++
Incremento
x++ → x = x + 1
--
Decremento
x-- → x = x - 1

Relacionais — comparam valores

>
Maior que
5 > 3 → 1 (true)
<
Menor que
2 < 8 → 1 (true)
==
Igual a
3 == 3 → 1 (true)
!=
Diferente de
4 != 5 → 1 (true)
>=
Maior ou igual
5 >= 5 → 1 (true)
<=
Menor ou igual
2 <= 9 → 1 (true)
⚠️
= vs == — Em C, = atribui um valor. == compara. Confundir os dois é um dos erros mais comuns para iniciantes!

Lógicos — combinam condições

&&
E lógico (AND)
true só se ambos forem true
||
OU lógico (OR)
true se pelo menos um for true
!
NÃO lógico (NOT)
inverte: !true → false
operadores.c 
#include <stdio.h>

int main() {
    int a = 10, b = 3;

    /* Aritméticos */
    printf("%d + %d = %d\n", a, b, a+b);  // 13
    printf("%d %% %d = %d\n", a, b, a%b);  // 1 (resto)

    /* Relacionais — retornam 0 ou 1 */
    printf("a > b: %d\n", a > b);    // 1 (verdadeiro)
    printf("a == b: %d\n", a == b);  // 0 (falso)

    /* Lógicos */
    printf("a>5 && b<5: %d\n", a>5 && b<5);  // 1
    printf("!(a == b): %d\n", !(a==b));       // 1

    /* Incremento */
    a++;
    printf("a após ++: %d\n", a);  // 11

    return 0;
}

03 — Decisões

Estruturas Condicionais

O programa escolhe um caminho baseado em uma condição. Se a condição for verdadeira (≠ 0) → executa um bloco; caso contrário → executa outro.

if
"Se" — executa o bloco somente se a condição for verdadeira.
else
"Senão" — executado quando nenhum if anterior foi verdadeiro.
else if
Encadeia múltiplas condições, testadas em ordem.
switch
Compara uma variável com vários valores fixos (case).
break
Encerra o switch (ou laço) imediatamente.
default
Bloco do switch executado quando nenhum case bate.

Fluxo do if / else if / else

INÍCIO
nota >= 7 ?
SIM
"Aprovado"
NÃO
nota >= 5 ?
SIM
"Recuperação"
NÃO
"Reprovado"
if_else.c 
#include <stdio.h>

int main() {
    float nota;
    printf("Digite sua nota (0-10): ");
    scanf("%f", &nota);

    if (nota >= 7.0f) {
        printf("✅ Aprovado!\n");
    } else if (nota >= 5.0f) {
        printf("⚠️  Recuperação.\n");
    } else {
        printf("❌ Reprovado.\n");
    }

    return 0;
}

switch — múltiplas opções fixas

💡
Quando usar switch? — Prefira switch quando você compara uma variável com valores inteiros ou char fixos. Fica mais legível do que vários else if.
switch.c 
#include <stdio.h>

int main() {
    int dia;
    printf("Dia da semana (1-7): ");
    scanf("%d", &dia);

    switch (dia) {
        case 1:
            printf("Domingo\n");
            break;
        case 2:
            printf("Segunda-feira\n");
            break;
        case 7:
            printf("Sábado\n");
            break;
        default:
            printf("Dia de semana comum.\n");
    }

    return 0;
}
🔴
Esqueceu o break? — Sem break, o C continua executando os case seguintes automaticamente (fall-through). Às vezes intencional; geralmente é um bug!
04 — Repetição

Laços (Loops)

Laços repetem um bloco de código enquanto uma condição for verdadeira. Evitam copiar e colar o mesmo código muitas vezes.

for
Usado quando o número de repetições é conhecido. Possui inicialização, condição e incremento na mesma linha.
while
Repete enquanto a condição for verdadeira. Testa antes de executar — pode nunca rodar.
do…while
Executa o bloco ao menos uma vez, depois testa a condição.
break
Sai do laço imediatamente, antes de a condição ser falsa.
continue
Pula para a próxima iteração sem executar o restante do bloco.
Variável de controle
Variável (geralmente i, j) que conta as iterações.

for — número de voltas conhecido

Sintaxe: for (inicialização; condição; incremento) { ... }

for (i = 0; i < 5; i++)
i = 0 0 < 5 ✓ → printf("%d", 0) → executa
i = 1 1 < 5 ✓ → printf("%d", 1) → executa
i = 2 2 < 5 ✓ → printf("%d", 2) → executa
i = 3 3 < 5 ✓ → printf("%d", 3) → executa
i = 4 4 < 5 ✓ → printf("%d", 4) → executa
i = 5 5 < 5 ✗ → PARA
for.c 
#include <stdio.h>

int main() {

    /* ── Contagem de 0 a 4 ── */
    for (int i = 0; i < 5; i++) {
        printf("%d ", i);       // 0 1 2 3 4
    }
    printf("\n");

    /* ── Soma de 1 a 100 ── */
    int soma = 0;
    for (int i = 1; i <= 100; i++) {
        soma += i;
    }
    printf("Soma 1-100: %d\n", soma);  // 5050

    /* ── Contagem regressiva ── */
    for (int i = 10; i >= 0; i--) {
        printf("%d ", i);
    }
    printf("🚀\n");

    return 0;
}

while — número de voltas indefinido

Sintaxe: while (condição) { ... } — a condição é testada antes de cada iteração.

while.c 
#include <stdio.h>

int main() {

    /* ── Dobrar número até passar de 1000 ── */
    int n = 1;
    while (n <= 1000) {
        printf("%d\n", n);
        n *= 2;   // n = n * 2
    }

    /* ── Validar entrada do usuário ── */
    int senha;
    while (1) {              // laço "infinito"
        printf("Digite a senha: ");
        scanf("%d", &senha);
        if (senha == 1234) break;   // sai quando acertar
        printf("Errado! Tente novamente.\n");
    }
    printf("Acesso liberado!\n");

    return 0;
}

do…while — executa pelo menos uma vez

Sintaxe: do { ... } while (condição); — o bloco roda antes de checar a condição.

do_while.c 
#include <stdio.h>

int main() {

    int opcao;

    do {
        printf("\n=== MENU ===\n");
        printf("1. Jogar\n");
        printf("2. Configurações\n");
        printf("0. Sair\n");
        printf("Escolha: ");
        scanf("%d", &opcao);

        if (opcao == 1) printf("Iniciando jogo...\n");
        if (opcao == 2) printf("Abrindo configs...\n");

    } while (opcao != 0);   // repete até 0

    printf("Até logo!\n");
    return 0;
}

break & continue — controlando o fluxo interno

break_continue.c 
#include <stdio.h>

int main() {

    /* continue — pula números pares */
    printf("Ímpares de 1-10: ");
    for (int i = 1; i <= 10; i++) {
        if (i % 2 == 0) continue;  // pula par
        printf("%d ", i);
    }
    printf("\n");    // 1 3 5 7 9

    /* break — acha primeiro múltiplo de 7 */
    for (int i = 1; i <= 100; i++) {
        if (i % 7 == 0) {
            printf("Primeiro múltiplo de 7: %d\n", i);
            break;    // não precisa continuar
        }
    }

    return 0;
}
Laço Testa condição Quando usar Execuções mínimas
for antes Quantidade conhecida 0
while antes Quantidade desconhecida 0
do…while depois Precisa executar ao menos 1x 1
05 — Pratique

Exercícios

A melhor forma de aprender é codando. Tente resolver cada exercício antes de ver a dica!

EX 01 Calculadora Simples
  • Leia dois números inteiros do usuário.
  • Leia o operador (+, -, *, /).
  • Exiba o resultado correto usando switch.
  • Trate a divisão por zero com uma mensagem de erro.
💡 Dica: use char op; scanf(" %c", &op); — o espaço antes de %c ignora o Enter anterior.
EX 02 Tabuada Interativa
  • Peça um número ao usuário.
  • Use um laço for para exibir a tabuada desse número de 1 a 10.
  • Formato: 5 x 3 = 15
💡 Dica: variáveis de controle no for podem ter qualquer nome — aqui, tente for (int mult = 1; mult <= 10; mult++).
EX 03 Detector de Par ou Ímpar + Positivo ou Negativo
  • Leia um número inteiro.
  • Usando if/else, diga se é par ou ímpar.
  • Usando outro if/else, diga se é positivo, negativo ou zero.
💡 Dica: par → n % 2 == 0. Combine condições com && para reduzir código.
EX 04 Contador de Números Primos
  • Leia um número limite N.
  • Use um for externo de 2 até N.
  • Para cada número, use um for interno para verificar se é primo.
  • Conte e exiba quantos primos existem até N.
💡 Dica: um número é primo se nenhum número de 2 até ele-1 o divide. Use uma variável int primo = 1; e break quando encontrar um divisor.
EX 05 Jogo Adivinhe o Número
  • Defina um número secreto no código: int secreto = 42;
  • Use um do…while para pedir palpites ao usuário.
  • A cada palpite, diga se é maior, menor ou igual ao secreto.
  • Conte quantas tentativas foram necessárias.
💡 Dica: do { ... } while (palpite != secreto); garante que o usuário tente ao menos uma vez.

Gabarito — Exercício 02 (tabuada)

tabuada.c 
#include <stdio.h>

int main() {
    int num;
    printf("Tabuada de qual número? ");
    scanf("%d", &num);

    printf("\n── Tabuada do %d ──\n", num);
    for (int mult = 1; mult <= 10; mult++) {
        printf("%d x %2d = %d\n", num, mult, num * mult);
    }

    return 0;
}
