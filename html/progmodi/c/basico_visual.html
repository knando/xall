01 — Dados

📦 Variável = Gaveta com Nome

Variável é como uma gaveta. Tem nome na frente e guarda um valor dentro.

🗄️
📛 idade
20
gaveta no mundo real
≡
💻

Cada gaveta guarda um valor:

📛 idade
🔢
20
int
📛 nota
📊
8.5
float
📛 letra
🔤
'A'
char
📛 nome
🪪
"Ana"
char[]
⚠️
Nome da gaveta — sem espaço, sem acento, começa com letra: ✅ nota_final   ❌ nota final   ❌ 1nota
💡
Trocar o valor da gaveta: idade = 21; — conteúdo antigo some. Gaveta guarda um valor apenas.
variaveis.c 
#include <stdio.h>

int main() {

    /* criar gavetas */
    int   idade    = 20;      // 🔢 número inteiro
    float nota     = 8.5f;   // 📊 número com vírgula
    char  letra    = 'A';    // 🔤 um caractere
    char  nome[30] = "Ana";  // 🪪 texto

    /* trocar o valor da gaveta */
    idade = 21;   // 20 some, entra 21

    return 0;
}

    int   idade = 20;
    float nota  = 8.5f;
    char  nome[30] = "Ana";

    printf("Nome: %s\n",   nome);   // 🪧 Nome: Ana

04 — Entrada

⌨️ scanf = Caixa de Correio

scanf é como uma caixa de correio. O usuário digita algo e o valor vai direto para dentro da gaveta.

👤
usuário
digita 25
➜
⌨️
teclado

➜
🗄️
gaveta

📍
endereço da casa
para saber onde entregar
=
💻
&idade
endereço da gaveta na memória
🔴
Sempre use & com scanf!   ✅ scanf("%d", &idade)    ❌ scanf("%d", idade)
Exceção: textos (char[]) — não precisam do &: scanf("%s", nome)
entrada.c 
#include <stdio.h>

int main() {
    int   idade;
    float nota;
    char  nome[50];

    printf("Seu nome: ");
    scanf("%s", nome);     // texto: sem &

    printf("Sua idade: ");
    scanf("%d", &idade);   // inteiro: com &

    printf("Sua nota: ");
    scanf("%f", &nota);    // float: com &

    printf("Olá, %s! %d anos, nota %.1f\n",
           nome, idade, nota);

    return 0;
}

06 — Decisões

🔀 if / else = Encruzilhada

O programa chega numa encruzilhada. Dependendo da resposta (✅ ou ❌), ele escolhe um caminho diferente.

🚶 Programa chega aqui
❓   nota >= 7 ?
✅ SIM
🎉 "Aprovado!"
❌ NÃO
❓   nota >= 5 ?
✅
⚠️ "Recuperação"
❌
😢 "Reprovado"
condicional.c 
#include <stdio.h>

int main() {
    float nota;
    printf("Nota: ");
    scanf("%f", &nota);

    if (nota >= 7) {                // ✅ SIM
        printf("Aprovado!\n");
    } else if (nota >= 5) {         // ⚠️ talvez
        printf("Recuperação\n");
    } else {                          // ❌ NÃO
        printf("Reprovado\n");
    }

    return 0;
}

switch = Cardápio de opções fixas

📋
Opção 1 → 🍕
Opção 2 → 🍔
Outro    → 🤷
cardápio do restaurante
=
switch (opcao) {
  case 1:
    printf("Pizza\n");
    break;
  case 2:
    printf("Hamburguer\n");
    break;
  default:
    printf("Outro\n");
}
🔴
Sempre escreva break; depois de cada case. Sem break o programa cai no próximo case automaticamente!
07 — Repetição

🔁 Laços = Esteira de Fábrica

Uma esteira repete o mesmo processo várias vezes. O laço faz o mesmo com o código — enquanto a condição for ✅.

🏭 for — voltas contadas

🔔
começo: i = 0
✅ enquanto i < 5
   fazer algo
   i++ (próxima)
❌ parar
campainha da fábrica
=
for (int i = 0; i < 5; i++) {
  // fazer algo 5 vezes
  printf("%d\n", i);
}
for (i = 0; i < 5; i++) — passo a passo
i = 00 < 5 ✅executa → imprime 0
i = 11 < 5 ✅executa → imprime 1
i = 22 < 5 ✅executa → imprime 2
i = 33 < 5 ✅executa → imprime 3
i = 44 < 5 ✅executa → imprime 4
i = 55 < 5 ❌ → PARA 🛑

🚦 while — repete enquanto for ✅

🚦
Enquanto sinal ✅ verde
→ avançar
Sinal ❌ vermelho
→ parar
=
while (n > 0) {
  printf("%d\n", n);
  n--;
}
repete enquanto n > 0

🍽️ do…while — executa pelo menos 1 vez

🍽️
Sempre sirvo o prato
→ depois pergunto:
"Quer mais?"
✅ → repete
❌ → para
=
do {
  printf("Senha: ");
  scanf("%d", &s);
} while (s != 1234);
tenta ao menos 1 vez
🏭
for  for (i=0; i<N; i++)
Sabe quantas vezes vai repetir  ·  número de voltas definido
🚦
while  while (cond)
Não sabe quantas vezes  ·  testa antes de entrar  ·  pode não executar nenhuma vez
🍽️
do…while  do { } while (cond);
Executa ao menos 1 vez  ·  testa depois de executar
loops.c 
#include <stdio.h>

int main() {

    /* 🏭 for — tabuada do 5 */
    for (int i = 1; i <= 10; i++) {
        printf("5 x %d = %d\n", i, 5*i);
    }

    /* 🚦 while — conta regressiva */
    int n = 5;
    while (n > 0) {
        printf("%d... ", n);
        n--;
    }
    printf("\n");

    /* 🍽️ do…while — validar senha */
    int senha;
    do {
        printf("Senha: ");
        scanf("%d", &senha);
    } while (senha != 1234);
    printf("Acesso!\n");

    return 0;
}
08 — Praticar

🏋️ Exercícios

Hora de criar! Tente resolver antes de ver a dica.

01 📦 Criar e exibir gavetas
  • Crie gavetas: seu nome, sua idade, sua nota favorita.
  • Use printf para mostrar tudo na tela.
  • Formato: "Nome: Ana | Idade: 20 | Nota: 9.5"
💡 Dica: use char nome[50] para texto, int para idade, float para nota.
02 ⌨️ Perguntar e responder
  • Pergunte o nome do usuário com printf.
  • Leia a resposta com scanf.
  • Exiba: "Olá, [nome]! Bem-vindo ao C!"
💡 Dica: texto sem espaços → scanf("%s", nome) sem &.
03 🔀 Estação do tempo
  • Peça a temperatura atual (número inteiro).
  • Se >= 30 → imprima "Muito quente!"
  • Se >= 20 → imprima "Temperatura boa!"
  • Se < 20 → imprima "Frio!"
💡 Dica: use if / else if / else — 3 caminhos diferentes na encruzilhada.
04 🏭 Esteira: tabuada
  • Pergunte um número ao usuário.
  • Use for para imprimir a tabuada de 1 a 10.
  • Formato: 5 x 3 = 15
💡 Dica: for (int i = 1; i <= 10; i++) — a esteira roda 10 vezes.
05 🍽️ Adivinhe o número
  • Defina um número secreto: int secreto = 42;
  • Use do…while para pedir palpites.
  • Se palpite > secreto → "Menor!"
  • Se palpite < secreto → "Maior!"
  • Acertou → "Parabéns! Em [X] tentativas!"
💡 Dica: use uma gaveta int tentativas = 0 e incremente com tentativas++ a cada volta.

✅ Gabarito — Exercício 03 (estação do tempo)

temperatura.c 
#include <stdio.h>

int main() {
    int temp;

    printf("Temperatura: ");
    scanf("%d", &temp);

    if (temp >= 30) {
        printf("Muito quente!\n");
    } else if (temp >= 20) {
        printf("Temperatura boa!\n");
    } else {
        printf("Frio!\n");
    }

    return 0;
}
