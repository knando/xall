VisuAlg → C

Você já sabe lógica de programação. Agora veja como tudo que aprendeu no VisuAlg se traduz diretamente para a linguagem C — lado a lado, sem surpresas.

algoritmo … fimalgoritmo int main() { … } escreva / leia printf / scanf
role para baixo
00 — Antes de começar

O que muda — e o que não muda

A lógica é a mesma. O que muda é a sintaxe — o jeito de escrever. Se você já faz programas no VisuAlg, já pensa como programador. Só precisa aprender o novo vocabulário.

O que NÃO muda
Variáveis, operadores, se/então, para/enquanto, lógica de decisão — tudo isso existe em C com o mesmo propósito.
O que muda
Palavras-chave em inglês, tipos de dados com nomes diferentes, ponto-e-vírgula obrigatório no final de cada instrução.
Compilado vs Interpretado
VisuAlg interpreta linha a linha. C é compilado — você gera um executável antes de rodar. Isso o torna muito mais rápido.
Case-sensitive
Em C, Nota e nota são variáveis diferentes. VisuAlg não diferencia maiúsculas de minúsculas.
Ponto-e-vírgula
Toda instrução em C termina com ;. Em VisuAlg isso não existe. Esquecer o ; é o erro nº 1 de iniciantes.
Blocos com { }
VisuAlg usa palavras como fimse, fimpara. C usa chaves { } para delimitar blocos de código.
VisuAlg / Portugol C Observação
algoritmo "nome" — (não existe) Em C não há declaração de nome de algoritmo.
var … inicio … fimalgoritmo int main() { … return 0; } O corpo do programa fica dentro de main().
escreva / escreval printf() escreval = printf com \n no final.
leia scanf() Em C é preciso informar o tipo com %d, %f, %s…
inteiro int Números inteiros.
real float / double Números com vírgula (ponto em C).
caractere char[] Texto em C é um array de char.
logico int (0 ou 1) C não tem tipo bool nativo (C89/C90). Use int ou #include <stdbool.h>.
<- (atribuição) = (atribuição) Em C o sinal de igual atribui valor.
{ } (conjunto de valores) { } (bloco de código) Chaves delimitam blocos em C.
// ou { comentário } // ou /* */ Comentários de linha e de bloco.
01 — Esqueleto

Estrutura do Programa

Todo programa em VisuAlg começa com algoritmo e termina com fimalgoritmo. Em C, tudo vive dentro da função main().

Estrutura mínima — "Olá, Mundo!"
VisuAlg
algoritmo "OlaMundo"

var
  { variáveis aqui }

inicio

  escreval("Olá, Mundo!")

fimalgoritmo
C
#include <stdio.h>

int main() {

  /* variáveis aqui */

  printf("Olá, Mundo!\n");

  return 0;
}
algoritmo "nome"
Cabeçalho obrigatório no VisuAlg. Define o nome do programa. Em C isso não existe — o arquivo .c tem o nome.
#include <stdio.h>
Inclui a biblioteca de entrada/saída padrão — necessária para usar printf e scanf. Equivale a "ligar" funções prontas.
var … inicio
Em VisuAlg variáveis ficam numa seção separada. Em C são declaradas dentro de qualquer bloco de código, antes de serem usadas.
return 0;
Encerra main() devolvendo 0 ao sistema operacional. Zero significa "programa terminou com sucesso".
escreval vs printf
escreval já pula linha automaticamente. printf não — você precisa colocar \n no final da string.
fimalgoritmo vs }
A chave de fechamento } encerra o bloco main(), assim como fimalgoritmo encerra o algoritmo.
🔵
Regra de ouro: sempre que você ver fimXXX no VisuAlg, em C ele vira uma chave }. E sempre que fechar um bloco no VisuAlg com uma palavra-chave, em C você fecha com }.
02 — Dados

Variáveis & Tipos

Os tipos de VisuAlg têm nomes em português. Em C os nomes são em inglês e existem variações de precisão e tamanho.

VisuAlg C Formato printf Diferença importante
inteiro int %d Idêntico em propósito. Em C tem 4 bytes (−2 bilhões a +2 bilhões).
real float %f VisuAlg usa vírgula; C usa ponto: 3.14 não 3,14.
real (mais preciso) double %lf Prefira double para cálculos científicos. Ocupa 8 bytes.
caractere char[] %s Em C texto é array de char com \0 no fim. Não é um tipo simples.
logico int (0 / 1) %d Sem tipo bool nativo em C89. Use stdbool.h para ter true/false.

Declarando variáveis

Declaração e atribuição de variáveis
VisuAlg
var
  idade    : inteiro
  nota     : real
  nome     : caractere
  aprovado : logico

inicio
  idade    <- 20
  nota     <- 8.5
  nome     <- "Ana"
  aprovado <- verdadeiro
C
/* dentro de main() */

  int   idade    = 20;
  float nota     = 8.5f;
  char  nome[50] = "Ana";
  int   aprovado = 1;   // 1 = verdadeiro

  /* não há seção var separada */
⚠️
Vírgula vs Ponto: em algumas versões do VisuAlg números reais usam vírgula (8,5). Em C usam ponto (8.5f). Escrever 8,5 em C é um erro de sintaxe.
🔵
logico em C: o valor 0 é falso, qualquer coisa diferente de zero é verdadeiro. Para ter true e false explícitos, adicione #include <stdbool.h> e use o tipo bool.
03 — Expressões

Operadores

Os operadores aritméticos são quase idênticos. Os lógicos e relacionais mudam de palavra para símbolo.

Categoria VisuAlg C Exemplo em C
Aritméticos
Adição + + a + b
Subtração - - a - b
Multiplicação * * a * b
Divisão / / a / b (inteiro ÷ inteiro = inteiro!)
Resto (mod) MOD % 10 % 3 → 1
Potência ^ pow(b, e) pow(2,3) → 8.0 — requer math.h
Relacionais
Igual = (comparação) == if (a == b) — DOIS sinais de igual!
Diferente <> != if (a != b)
Maior que > > if (a > b)
Menor que < < if (a < b)
Maior ou igual >= >= if (a >= b)
Menor ou igual <= <= if (a <= b)
Lógicos
E lógico e && if (a > 0 && b > 0)
OU lógico ou || if (a == 0 || b == 0)
NÃO lógico nao ! if (!aprovado)
Atribuição
Atribuição simples <- = x = 5;
Incrementar x <- x + 1 x++ ou x += 1 Atalho muito usado em laços.
Decrementar x <- x - 1 x-- ou x -= 1 Conta regressiva.
🔴
Armadilha clássica — = vs ==
No VisuAlg = em condição compara. Em C, = atribui e == compara.
if (x = 5) → sempre verdadeiro (atribui 5 e testa se 5 != 0)!
if (x == 5) → compara x com 5. ✓
Operadores na prática
VisuAlg
var
  a, b, res : inteiro

inicio
  a <- 10
  b <- 3

  res <- a + b
  escreval(res)          { 13 }

  res <- a MOD b
  escreval(res)          { 1 }

  res <- a ^ b
  escreval(res)          { 1000 }
fimalgoritmo
C
#include <stdio.h>
#include <math.h>   // para pow()

int main() {
  int a = 10, b = 3, res;

  res = a + b;
  printf("%d\n", res);  // 13

  res = a % b;
  printf("%d\n", res);  // 1

  printf("%.0f\n", pow(a,b)); // 1000
  return 0;
}
04 — Comunicação

Entrada & Saída

escreva / leia são substituídos por printf / scanf. A grande diferença: em C é preciso dizer o tipo do dado com especificadores de formato.

%d
Inteiro (int). Usa-se tanto no printf quanto no scanf.
%f
Float no printf. No scanf também lê float. Para double no scanf use %lf.
%s
String (char[]). scanf("%s") para no primeiro espaço. Use fgets para frases.
%c
Um único caractere (char). Dica: coloque espaço antes — " %c" — para ignorar o Enter anterior.
& no scanf
O & passa o endereço da variável. Sem ele o scanf não consegue gravar o valor. Exceção: strings (char[]) já são endereços.
\n no printf
escreval pula linha automaticamente. printf não — acrescente \n ao final da string para quebrar a linha.
Entrada e saída de dados
VisuAlg
var
  nome  : caractere
  idade : inteiro
  nota  : real

inicio
  escreva("Nome: ")
  leia(nome)

  escreva("Idade: ")
  leia(idade)

  escreva("Nota: ")
  leia(nota)

  escreval("Olá, ", nome)
  escreval("Idade: ", idade)
  escreval("Nota:  ", nota:1:2)
fimalgoritmo
C
#include <stdio.h>
#include <string.h>

int main() {
  char  nome[50];
  int   idade;
  float nota;

  printf("Nome: ");
  fgets(nome, sizeof(nome), stdin);
  nome[strcspn(nome,"\n")] = '\0';

  printf("Idade: ");
  scanf("%d", &idade);

  printf("Nota: ");
  scanf("%f", &nota);

  printf("Olá, %s\n", nome);
  printf("Idade: %d\n", idade);
  printf("Nota:  %.2f\n", nota);
  return 0;
}
💡
Formatando números no printf: %.2f é o equivalente ao nota:1:2 do VisuAlg — exibe o float com 2 casas decimais. A sintaxe é %[largura].[casas]f.
05 — Decisões

Estruturas Condicionais

se … entao … senao … fimse vira if … else com chaves. A lógica é exatamente a mesma.

se / senao → if / else

Verificar aprovação do aluno
VisuAlg
var
  nota : real
inicio
  leia(nota)

  se (nota >= 7) entao
    escreval("Aprovado")
  senao
    se (nota >= 5) entao
      escreval("Recuperação")
    senao
      escreval("Reprovado")
    fimse
  fimse
fimalgoritmo
C
#include <stdio.h>
int main() {
  float nota;
  scanf("%f", &nota);

  if (nota >= 7) {
    printf("Aprovado\n");
  } else if (nota >= 5) {
    printf("Recuperação\n");
  } else {
    printf("Reprovado\n");
  }

  return 0;
}
🔵
fimse → } — Cada fimse do VisuAlg corresponde a uma chave } de fechamento em C. senao se vira else if (tudo numa linha só, sem chave entre eles).

escolha / caso → switch / case

Menu de opções
VisuAlg
var
  op : inteiro
inicio
  leia(op)

  escolha (op)
    caso 1:
      escreval("Novo jogo")
    caso 2:
      escreval("Carregar")
    outrocaso:
      escreval("Inválido")
  fimescolha
fimalgoritmo
C
#include <stdio.h>
int main() {
  int op;
  scanf("%d", &op);

  switch (op) {
    case 1:
      printf("Novo jogo\n");
      break;    // VisuAlg não precisa
    case 2:
      printf("Carregar\n");
      break;
    default:
      printf("Inválido\n");
  }

  return 0;
}
🔴
break obrigatório no switch! — No VisuAlg o escolha/caso para automaticamente em cada caso. Em C, sem o break, o programa continua executando os casos seguintes (fall-through). Sempre adicione break; ao fim de cada case.
VisuAlg C
se (cond) entaoif (cond) {
senao} else {
senao se (cond) entao} else if (cond) {
fimse}
escolha (var)switch (var) {
caso valor:case valor: … break;
outrocaso:default:
fimescolha}
06 — Repetição

Estruturas de Laço

Os três laços do VisuAlg têm equivalentes diretos em C. A lógica não muda — apenas a sintaxe.

para … faca → for

Contar de 1 a 10 e somar
VisuAlg
var
  i, soma : inteiro
inicio
  soma <- 0

  para i de 1 ate 10 faca
    escreval(i)
    soma <- soma + i
  fimpara

  escreval("Soma: ", soma)
fimalgoritmo
C
#include <stdio.h>
int main() {
  int i, soma = 0;

  for (i = 1; i <= 10; i++) {
    printf("%d\n", i);
    soma += i;
  }

  printf("Soma: %d\n", soma);
  return 0;
}
💡
Anatomia do for em C:
for (i = 1 ; i <= 10 ; i++)
↑ inicio        ↑ condição      ↑ passo
É exatamente o para i de 1 ate 10 do VisuAlg — só que tudo numa linha com ponto-e-vírgula separando as partes.
⚠️
Passo diferente de 1: no VisuAlg você usa passo 2. Em C, basta mudar o incremento: i += 2 ou i = i + 2 no terceiro campo do for.

enquanto … faca → while

Ler até digitar 0
VisuAlg
var
  n : inteiro
inicio
  n <- -1

  enquanto (n <> 0) faca
    escreva("Digite (0 p/ sair): ")
    leia(n)
    escreval("Você digitou: ", n)
  fimenquanto

  escreval("Fim!")
fimalgoritmo
C
#include <stdio.h>
int main() {
  int n = -1;

  while (n != 0) {
    printf("Digite (0 p/ sair): ");
    scanf("%d", &n);
    printf("Você digitou: %d\n", n);
  }

  printf("Fim!\n");
  return 0;
}

repita … ate → do … while

Validar entrada — executa ao menos 1 vez
VisuAlg
var
  senha : inteiro
inicio

  repita
    escreva("Senha: ")
    leia(senha)
  ate (senha = 1234)

  escreval("Acesso liberado!")
fimalgoritmo
C
#include <stdio.h>
int main() {
  int senha;

  do {
    printf("Senha: ");
    scanf("%d", &senha);
  } while (senha != 1234);

  printf("Acesso liberado!\n");
  return 0;
}
⚠️
Condição invertida! — No VisuAlg repita … ate (cond) para quando a condição for verdadeira. Em C, do … while (cond) continua enquanto a condição for verdadeira. Para traduzir, inverta a condição: ate (senha = 1234) → while (senha != 1234).
VisuAlg C Detalhe crítico
para i de 1 ate N faca for (i = 1; i <= N; i++) Passo padrão = 1. Mude o terceiro campo para outro passo.
para i de N ate 1 passo -1 faca for (i = N; i >= 1; i--) Contagem regressiva: inverte condição e usa i--.
fimpara } Chave fecha o bloco do for.
enquanto (cond) faca while (cond) { Mesma condição, sem alteração.
fimenquanto } Chave fecha o bloco do while.
repita do { Bloco executado pelo menos 1 vez.
ate (cond) } while (!cond); ATENÇÃO: condição invertida! VisuAlg para quando cond é V; C continua quando cond é V.
07 — Pratique a Migração

Exercícios

Em cada exercício você receberá um algoritmo em VisuAlg. Sua tarefa é reescrevê-lo em C produzindo o mesmo resultado.

EX 01 Migrar: Calculadora de IMC
VisuAlg — traduzir este código
var
  peso, altura, imc : real
inicio
  escreva("Peso (kg): ")
  leia(peso)
  escreva("Altura (m): ")
  leia(altura)

  imc <- peso / (altura * altura)
  escreval("IMC: ", imc:1:2)

  se (imc < 18.5) entao
    escreval("Abaixo do peso")
  senao
    se (imc < 25) entao
      escreval("Peso normal")
    senao
      escreval("Sobrepeso")
    fimse
  fimse
fimalgoritmo
  • Declare as variáveis como float ou double.
  • Use scanf("%f", &peso) para ler os valores.
  • Lembre-se: imc:1:2 em VisuAlg vira %.2f em C.
  • Traduza os se/senao/fimse para if/else if/else com chaves.
💡 Dica: para calcular altura * altura você pode usar a expressão diretamente ou pow(altura, 2) com #include <math.h>.
EX 02 Migrar: Tabuada com para/faca
VisuAlg — traduzir este código
var
  n, i : inteiro
inicio
  escreva("Tabuada de: ")
  leia(n)
  para i de 1 ate 10 faca
    escreval(n, " x ", i, " = ", n*i)
  fimpara
fimalgoritmo
  • Traduza o para … ate … faca … fimpara para for (i=1; i<=10; i++) { }.
  • No printf, use um único printf com todos os valores: printf("%d x %d = %d\n", n, i, n*i).
💡 Dica: no VisuAlg você separa valores no escreval com vírgulas. Em C tudo vai num único printf com múltiplos especificadores de formato.
EX 03 Migrar: Adivinhe o número (enquanto)
  • No VisuAlg: declare secreto = 42, leia palpites em um enquanto (palpite <> secreto).
  • A cada tentativa, diga se o palpite é maior ou menor que o secreto.
  • Ao acertar, exiba o número de tentativas.
  • Traduza para C usando while, scanf e if/else.
💡 Dica: <> no VisuAlg vira != em C. Declare um contador int tentativas = 0 e incremente com tentativas++ dentro do loop.
EX 04 Migrar: Menu com repita/ate e escolha/caso
  • Crie um menu com 3 opções + sair (0).
  • Primeiro escreva em VisuAlg usando repita … ate (op = 0) e escolha/caso.
  • Depois migre para C: converta repita/ate para do/while com condição invertida.
  • Converta escolha/caso para switch/case — não esqueça o break!
💡 Dica dupla: (1) ate (op = 0) → while (op != 0) — condição invertida! (2) Cada caso em C precisa de break; no final.

Gabarito — Exercício 01 (IMC em C)

imc.c 
#include <stdio.h>

int main() {
    float peso, altura, imc;

    printf("Peso (kg): ");
    scanf("%f", &peso);
    printf("Altura (m): ");
    scanf("%f", &altura);

    imc = peso / (altura * altura);
    printf("IMC: %.2f\n", imc);

    if (imc < 18.5f) {
        printf("Abaixo do peso\n");
    } else if (imc < 25.0f) {
        printf("Peso normal\n");
    } else {
        printf("Sobrepeso\n");
    }

    return 0;
}
